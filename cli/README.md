# unfocus-cli

Terminal break reminders for developers. Touch grass sometimes.

> **Coming soon.** This package is a placeholder for the upcoming CLI version of [Unfocus](https://github.com/akshat10/Unfocus-b1).

## What it will do

- Terminal-native break reminders with ASCII art
- Customizable intervals (20m, 30m, 45m, 60m)
- Break types: eyes, breath, posture, hands, hydration, window
- Beautiful terminal UI
